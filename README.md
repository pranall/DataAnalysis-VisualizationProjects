# DataAnalysis-VisualizationProjects
Hi, these are all the projects I did in Data Analysis and Visualization in Python.

-------

**Data MAnipulation and Basic Plots:** Processed the Urban Park Rangers dataset to analyze changes over time in animal assistance requests. Filtered and reshaped relevant columns into an importable table, calculated trends, and exported outputs to separate sheets. Consolidated findings into a publication table with informative captions

[Full Data MAnipulation and Basic Plots Project Link](https://github.com/pranall/DataAnalysis-VisualizationProjects/tree/main/Data%20Manipulation%20%26%20Basic%20Plots)

**Plots, Flowcharts and Cognitive Load:** This project focuses on creating a Pretty Plot and an Ugly Plot using the Maricopa Covid19 dataset. Created a pulishable pretty plot and a non publishable Ugly plot and gave more insights on how to make visualization legible.

[Full Pretty and Ugly Plots project link](https://github.com/pranall/DataAnalysis-VisualizationProjects/tree/main/Plots%2C%20Flowcharts%20%26%20Cognitive%20Load)

**Advanced Plots and Illusions:** This project focuses on creating a Pretty Plot and an Illusion Plot using the Slugs and Snails dataset. Created a pulishable pretty plot and a non publishable Illusion plot and gave more insights on how to make visualization legible.

[Full Pretty and Illusion Plots project link](https://github.com/pranall/DataAnalysis-VisualizationProjects/tree/main/Advanced%20Plots%20and%20Illusions)
